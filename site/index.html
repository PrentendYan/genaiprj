Can LLMs Audit Quant Research?

A working benchmark and prototype for adversarial AI review of financial backtests, integrating DARF and CORAX design patterns.

The project evaluates whether adversarial review systems can catch lookahead bias, leakage, invalid time-series splits, missing trading costs, and unsupported performance claims in quant research workflows.

Integrated Systems

SystemRole in this project
DARFCross-model adversarial review using blind briefs and gate decisions.
CORAXCodex-on-Codex Santa Method review with a heterogeneous Sentinel for groupthink checks.

Working Artifact

Run python -m src.quant_audit_benchmark.cli --cases benchmark_cases/cases.json to reproduce the benchmark metrics. The harness uses labeled financial-code cases and a bundled BTC data fixture.

Evidence

The current offline version reports precision, recall, and F1 for three review profiles: a single-pass baseline, a DARF-inspired cross-model profile, and a CORAX-inspired Santa Method plus Sentinel profile.

Limitations

The deterministic harness is a reproducible scaffold. The next experiment should run live LLM reviewers, store raw outputs, and compare human labels, cost, latency, and failure modes.
