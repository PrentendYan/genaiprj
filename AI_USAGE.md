# AI Usage Statement

We used AI coding assistants to help design the project structure, draft documentation, and implement the deterministic benchmark harness.

AI tools helped with:

- Translating local DARF/CORAX design notes into a clean course-project artifact.
- Drafting the README, report, and audience-facing static page.
- Implementing and testing the offline benchmark harness.

Human checks included:

- Reviewing the project scope against the course rubric.
- Keeping secrets, API keys, local MCP state, and personal configuration files out of the submission.
- Verifying that the benchmark does not silently fall back to synthetic data.
- Running deterministic tests and inspecting generated metrics.

The AI-generated outputs are not treated as source truth. The labeled benchmark cases, final claims, and oral-defense explanations should be reviewed by the student team before submission.
